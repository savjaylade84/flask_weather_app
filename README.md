## Flask Weather App
### A python flask based website that uses openweathermap api to<br> 
### show weather of all the city around the world.

dependent packages

- flask
- requests
- python-dotenv

<br>

Website<br>
[Weather App Link](https://weatherflaskapp.onrender.com/)<br>

To Run The Code<br>
`source venv/bin/activate` <br>
`python3 app.py`
